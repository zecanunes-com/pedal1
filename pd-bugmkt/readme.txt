=== Description ===
Ao invéz de apresentar uma página de erro 404, envia o usuário do site para uma página de sua escolha, é a melhor maneira para problemas com os mecanismos de busca. Também possui envio de alertas por e-mail.

== Installation ==
1. Upload this plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress. 
3. Adjust settings

== Remove plugin ==
1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress
It's best to use the build in delete function of wordpress. 
That way all the stored data will be removed and no orphaned data will stay.
